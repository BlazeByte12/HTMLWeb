Šta je HTML?

HTML je osnovni jezik koji se koristi za pravljenje web stranica. Svaka stranica koju vidiš na internetu napravljena je pomoću HTML-a.

Pomoću HTML-a mi ne određujemo kako stranica izgleda, nego šta se na njoj nalazi – naslovi, tekst, slike, linkovi i drugi elementi.

Kako funkcioniše web stranica?

Kada u pretraživač (Google Chrome, Edge, Firefox…) upišeš neku adresu, tvoj računar šalje zahtjev serveru.

Server zatim vraća HTML dokument koji pretraživač prikazuje kao web stranicu.

HTML je zapravo tekstualni fajl koji pretraživač “pretvara” u vizuelnu stranicu.

Struktura URL adrese

URL je adresa web stranice.

On se sastoji od:

  • Protokola (https)
  • Domene (naziv stranice)
  • Putanje (tačna lokacija stranice)

Struktura domene

Domena je naziv web stranice.

Na primjer: www.mojaskola.com

  • www je prefiks
  • mojaskola je naziv
  • .com je ekstenzija (tip domene)

Osnovna struktura HTML dokumenta

Svaka HTML stranica ima osnovnu strukturu. Bez nje stranica ne može pravilno raditi. 

<!DOCTYPE html>
<html>
<head>
    <title>Naslov stranice</title>
</head>
<body>
    Sadržaj stranice
</body>
</html>

Ukratko:

  • <html> – označava početak HTML dokumenta
  • <head> – sadrži informacije o stranici
  • <body> – sadrži sadržaj koji vidimo

Zaključak

HTML je temelj svake web stranice. Kada naučiš osnove HTML-a, lakše ćeš razumjeti i druge web tehnologije.
