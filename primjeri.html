Primjeri korištenja HTML-a

Nakon što smo naučili osnovne HTML naredbe, možemo ih spojiti i napraviti jednostavnu web stranicu.

U nastavku možeš vidjeti jedan mali primjer kako HTML elementi zajedno čine jednu web stranicu.

Primjer jednostavne stranice

<h1>Moja web stranica</h1>

<p>Dobrodošli na moju prvu web stranicu.</p>

<img src="slika.jpg" alt="Primjer slike">

<a href="https://example.com">Posjeti stranicu</a>

Ovdje vidimo kombinaciju naslova, paragrafa, slike i linka – osnovnih elemenata svake web stranice.

Mali zadatak

Pokušaj napraviti svoju HTML stranicu koja sadrži:

  • jedan naslov
  • kratki paragraf
  • jednu sliku
  • jedan link

Sačuvaj fajl kao index.html i otvori ga u browseru da vidiš kako izgleda.
